## How To Run

1. ``git clone https://github.com/mukhoplus/rank-system.git``
2. ``cd ./src/main/``
3. ``git clone https://github.com/mukhoplus/rank-system_front.git frontend``
4. Set database tables.
5. Run spring boot server.
